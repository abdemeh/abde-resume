## Comment faire tourner ce chef-d'œuvre chez vous ?

Vous voulez voir comment c'est fait ? Vous voulez copier mon code ? Pas de souci, voici la recette :

1. **Cloner le repo** (volez-le, légalement) :

    ```bash
    git clone https://github.com/abdemeh/abde-resume.git
    ```

2. **Installer les dépendances** (télécharger tout internet) :

    ```bash
    npm install
    ```

3. **Lancer le serveur de dev** (et prier pour que ça marche) :

    ```bash
    npm run dev
    ```

4. **Admirer le résultat** sur `http://localhost:5173`.
